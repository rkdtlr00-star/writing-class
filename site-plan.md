# 개인 브랜딩 + 홍보 사이트 기획 문서

> 카피라이터의 글쓰기 과외 — 개인 브랜딩 & 포트폴리오 사이트

---

## 1. 사이트 개요

| 항목 | 내용 |
|------|------|
| 사이트 목적 | 초등 글쓰기 1:1 과외 홍보 + 개인 브랜딩 |
| 타겟 방문자 | 초등 3~4학년 자녀를 둔 학부모 (양천구·목동 인근) |
| 핵심 메시지 | "카피라이터가 가르치는, 재미있고 특별한 글쓰기" |
| 디자인 톤 | 미니멀 / 따뜻한 오프화이트 계열 / 손글씨 느낌의 포인트 폰트 혼용 |

---

## 2. 사이트 구조 (페이지 구성)

```
/ (홈)
├── Hero — 인트로 + 핵심 카피
├── About — 선생님 소개
├── Why — 이 수업이 다른 이유
├── Curriculum — 5주 커리큘럼
├── Portfolio — 수업 장면 / 결과물 (추후 추가)
└── Contact — 수강 신청 및 문의
```

---

## 3. 섹션별 콘텐츠 상세

---

### 3-1. Hero (첫 화면)

**헤드라인 (대안 3가지 — 하나 선택)**

```
A. 생각이 가지처럼 뻗어나가는 글쓰기 수업
B. 일기도 독후감도 아닌, 색다른 글쓰기의 세계
C. 카피라이터가 전하는 5주간의 글쓰기 이야기
```

**서브카피**

```
글과 아직 친하지 않아도 괜찮아요.
놀고 떠들다 보면 어느새 글이 써지는 수업입니다.
```

**CTA 버튼**

```
[수업 더 알아보기]   [수강 문의하기]
```

### 3-2. About (선생님 소개)

**섹션 타이틀**

```
안녕하세요, 글 쓰는 걸 좋아하게 된 카피라이터입니다.
```

**본문**

```
국어국문학과를 나와 카피라이터로 일하고 있는 제가,
어릴 적 글쓰기를 그다지 좋아하지 않았다면 믿어지시나요?

많은 아이들이 그렇듯 저의 글쓰기 역사도 일기와 독후감 숙제에서 출발했습니다.
'참 잘했어요' 도장을 받기 위해 연필을 쥐었던 그 시절,
빈 공책은 학교 운동장보다도 드넓어 보였죠.

이 수업은 어린 시절의 저를 떠올리며 기획했습니다.
글쓰기가 학습 스킬을 넘어, 나를 나답게 표현하고 성장시킬
든든한 힘이 된다는 걸 함께 발견하는 시간이 될 거예요.
```

**약력 (타임라인 형태)**

```
🎓  이화여자대학교 국어국문학과 졸업
✏️  커머스 플랫폼 콘텐츠 에디터
📣  하나은행 콘텐츠 카피라이터
📝  현재 — 카피라이터 & 초등 글쓰기 과외
```

### 3-3. Why (이 수업이 다른 이유)

**섹션 타이틀**

```
학원에서는 절대 경험할 수 없는 수업입니다
```

**카드형 포인트 4가지**

| 아이콘 | 제목 | 설명 |
|--------|------|------|
| ✍️ | 카피라이터의 글 감각 | 평범한 것을 매력적으로 표현하는 카피라이팅 기법을 아이 눈높이에 맞춰 전달합니다 |
| 💬 | 친구처럼 대화하는 수업 | 워밍업 활동으로 아이와 충분히 떠들며 자연스럽게 생각과 표현을 끌어냅니다 |
| 🌱 | 틀려도 혼내지 않아요 | 맞춤법은 수업 끝에 조용히 교정합니다. 자유롭게 표현하는 즐거움을 먼저 경험합니다 |
| 📖 | 문학 + 카피의 결합 | 국문학에서 배운 문학의 매력과 카피라이터로 익힌 문장 감각을 아낌없이 나눕니다 |

### 3-4. Curriculum (5주 커리큘럼)

**섹션 타이틀**

```
5주 뒤, 아이의 문장에 어떤 변화가 찾아올까요?
```

**커리큘럼 테이블**

| 회차 | 주제 | 핵심 활동 |
|------|------|-----------|
| 1주차 | 나는 꼬마 카피라이터! | 왓츠인마이백 + 카피라이팅 감각 깨우기 + 나만의 상품 카탈로그 만들기 |
| 2주차 | 내 마음을 뾰족하게 표현하는 법 | 감정 단어 탐구 + 다채로운 표현 익히기 + 감정 시 쓰기 |
| 3주차 | 세상에 하나뿐인 나 사용 설명서 | 관찰 훈련 + 나 탐구 활동지 + 나만의 사용 설명서 완성 |
| 4주차 | 엄마·아빠로 빙의 완료! | 소설 발췌독 + 타인의 시점 상상하기 + 부모님 시점 일기 쓰기 |
| 5주차 | 나만의 이야기 완성 | 마무리 글쓰기 + 수료증 수여 |

### 3-5. 수업 정보 (Info)

```
대상     초등 3~4학년 (글쓰기와 친해지고 싶은 아이)
일정     주 1회 · 총 5회
시간     1회 1시간 20분
수업료   총 35만원
장소     학생 집 방문 (어려운 경우 인근 스터디카페 협의)
지역     신정역 · 목동역 · 신정네거리역 · 양천구청역 인근
```

### 타이포그래피


### 레이아웃 원칙

- 여백을 충분히 사용 (콘텐츠 밀도 낮게)
- 긴 텍스트는 짧은 단락으로 분리
- 모바일 우선 설계 (학부모가 스마트폰으로 접근할 가능성 높음)
- 수업 분위기가 느껴지는 일러스트 또는 손그림 요소 1~2개 삽입

## 5. 기술 스택 추천

| 방식 | 도구 | 적합한 경우 |
|------|------|-------------|
| **추천 — 빠른 배포** | Notion + Super.so 또는 Framer | 코딩 없이 미니멀하게 완성, 수정 쉬움 |
| 중급 | 순수 HTML/CSS + GitHub Pages | 가볍고 무료, 기본 코딩 필요 |
| 고급 | Next.js + Tailwind + Vercel | SEO 최적화, 장기적 확장성 |

> **현재 단계 추천:** Framer 또는 Notion 기반으로 빠르게 런칭 후, 수업 경험이 쌓이면 전환
